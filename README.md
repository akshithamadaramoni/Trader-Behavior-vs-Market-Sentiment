# Trader-Behavior-vs-Market-Sentiment
“Analysis of trader behavior vs market sentiment in cryptocurrency using Python, EDA, and ML.”
## Project Overview
Analyzed how market sentiment impacts cryptocurrency trader behavior. Explored Fear & Greed Index, historical trade data, and applied a ML model to predict trade direction.

## Files
- `Trader Behavior vs Market Sentiment.docx` → Project documentation
- `Trader_Analysis.ipynb` → Jupyter notebook with code and analysis
- `images/` → Charts and plots used in analysis (optional)

## How to Run
1. Open `Trader_Analysis.ipynb` in Jupyter Notebook or VS Code.
2. Follow the notebook steps to reproduce results.
